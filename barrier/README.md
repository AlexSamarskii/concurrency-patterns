**Barrier** — это синхронизационный примитив, который заставляет группу горутин ждать друг друга в определённой точке выполнения. Все горутины должны достичь барьера, прежде чем любая из них сможет продолжить работу. Это похоже на sync.WaitGroup, но с возможностью повторного использования для синхронизации нескольких фаз работы.

Пример из реальной жизни: несколько потоков обрабатывают части большого массива. После завершения каждой итерации они должны синхронизироваться, чтобы обменяться граничными значениями перед следующей итерацией.
