**Rendezvous** — это паттерн синхронизации двух (или более) горутин, при котором каждая из них достигает определённой точки выполнения и ждёт, пока другая также достигнет этой точки. После этого они могут обменяться данными и продолжить работу. По сути, это двусторонняя синхронизация с возможностью обмена сообщениями
