**Cancellation Pattern** — это способ передать сигнал завершения от одной горутины (обычно главной) к другим горутинам, чтобы они корректно освободили ресурсы и прекратили выполнение. В Go для этого идиоматически используется пакет context, который предоставляет унифицированный механизм отмены, дедлайнов и передачи значений, связанных с запросом
